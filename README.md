# ポケモン剣盾 ステータス検索・比較app

## 制作背景
オンライン対戦中に、制限時間内に相手 (自分) のポケモンのステータスを検索したいと思い作成

## 創意工夫
・データ数が多く、また早く検索したいため、オートコンプリート機能を実装  
・レスポンシブ対応

## 使用技術
Node.js (express) / MongoDB / Vue.js / Tailwind CSS / Ant Design Vue
## Requirement (バージョンなど)
  Node.js: v16.13.2.,  
  @vue/cli: 5.0.1,  
  tailwindcss: 3.0.23,  
  ant-design-vue: 2.2.8,  
  webpack: 5.70.0
